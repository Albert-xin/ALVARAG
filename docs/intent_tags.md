# 意图标签库设计

## 存储位置
- `docs/intent_tags.md`（精确配置）
- `Milvus向量库`（相似样本）

## 一、MD 精确标签（grep 可读、系统固定、不可变）
- `rag_query`：知识库检索问答（默认兜底）
- `chat`：日常闲聊
- `command`：系统指令操作
- `sensitive`：违规敏感内容
- `unknown`：未知意图

## 二、向量库相似样本（Milvus）
- 各类意图对应的用户典型问句向量
- 用于低置信度时语义匹配补正
- 向量字段：text, intent, score, create_time

## 三、结构规范
- 固定标签名、置信度阈值 0.7
- 样本按意图分类存储，支持增量更新
- 路由策略与标签一一绑定
- 低置信度（<0.7）强制使用向量检索相似意图修正
